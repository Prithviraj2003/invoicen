---
title: Environment Variables
description: Environment variables required for running Invoicen
---

| Key                             | Value                                           | Description                                      |
|---------------------------------|-------------------------------------------------|--------------------------------------------------|
| NEXT_PUBLIC_ANALYTICS_DOMAIN    | invoicen.keizerworks.com                        | Domain for analytics                              |
| NEXT_PUBLIC_ANALYTICS_FILE_URL  | https://plausible.internal.keizerworks.com      | URL for analytics file                            |
| PORT                            | 8080                                            | Port on which the application runs                |
| APP_ENV                         | local                                           | Application environment                           |
| DB_HOST                         | localhost                                       | Database host                                     |
| DB_PORT                         | 5432                                            | Database port                                     |
| DB_DATABASE                     | postgres                                        | Database name                                     |
| DB_USERNAME                     | postgres                                        | Database username                                 |
| DB_PASSWORD                     | password                                        | Database password                                 |
| DB_SCHEMA                       | public                                          | Database schema                                   |
| MINIO_ROOT_USER                 | minioadmin                                      | MinIO root user                                   |
| MINIO_ROOT_PASSWORD             | minioadmin                                      | MinIO root password                               |
| MINIO_PORT                      | 9000                                            | MinIO port                                        |
| REDIS_HOST                      | localhost                                       | Redis host                                        |
| REDIS_PORT                      | 6379                                            | Redis port                                        |
| REDIS_PASSWORD                  | password                                        | Redis password                                    |
| SMTP_HOST                       | localhost                                       | SMTP host                                         |
| SMTP_PORT                       | 1025                                            | SMTP port                                         |
| SMTP_USER                       | keizer                                          | SMTP user                                         |
| MAIL_FROM                       | auth@keizer.com                                 | Email address for outgoing mail                   |
